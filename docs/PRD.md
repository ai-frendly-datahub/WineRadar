# PRD – WineRadar MVP

## 목표
- 와인 산업 종사자가 매일 아침 5분 이내에 핵심 이슈를 파악하도록 돕는다.
- 최소 3개의 대표 소스(미디어, 수입사, 커뮤니티)를 커버한다.

## 사용자 스토리
1. **마케팅 담당자**로서, 주요 기사/입점 소식을 요약본으로 받고 싶다.
2. **수입사 대표**로서, 경쟁사/브랜드 언급이 어떻게 변하는지 알고 싶다.
3. **커뮤니티 운영자**로서, 커뮤니티 내 인기 토픽을 파악하고 싶다.

## 성공 지표 (초기)
- 일일 리포트 생성 성공률 95% 이상
- HTML 리포트 내 카드 최소 10개 이상
- 텔레그램 알림 클릭률 30% 이상 (도입 후 측정)

## 범위 (In)
- 수집/분석/리포트/푸시 파이프라인
- GitHub Actions + Pages 기반 자동 배포
- TDD 및 문서화

## 범위 (Out)
- 실시간 알림
- 고급 NLP (LLM 요약, 감성 분석 등)
- 유료 사용자 관리

## 타임라인
- Phase 0: 스켈레톤 (현재)
- Phase 1: 기본 파이프라인 완성
- Phase 2: 리포트 품질 향상, UI 개선

## 리스크
- 소스 구조 변경 → Collector 유지보수 비용 증가
- 텔레그램/이메일 제한 → 알림 채널 다변화 필요
- 그래프 저장소 용량 증가 → 별도 DB 전환 고려
