# AI 작업 템플릿

다음 체크리스트를 사용해 Codex/Claude 등의 AI 에이전트에게 작업을 지시하세요.

1. **요약**: 문제 배경과 기대 결과를 2~3줄로 설명합니다.
2. **상세 요구사항**: 필수 기능, 제약 조건, 입력/출력 형식을 bullet 로 정리합니다.
3. **테스트**: 통과해야 하는 테스트 수단(pytest, curl 등)을 명시합니다.
4. **검증 기준**: 코드 리뷰, 문서 업데이트, 성능/보안 요구 등 성공 판정을 위한 기준을 적습니다.
5. **추가 정보**: 참조 링크, 관련 이슈 번호, 샘플 데이터 경로 등을 제공합니다.

가능하면 “무엇을/왜”에 초점을 맞춰 작성하고, 구현 세부 사항은 에이전트가 스스로 설계할 수 있도록 여지를 주세요.
