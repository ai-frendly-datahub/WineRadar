# 데이터 소스 신뢰성 평가 (Source Reliability Assessment)

## 평가 일시
2025-11-19

## 평가 방법론

### 평가 기준 (5개 차원)

1. **기관 신뢰도 (Institutional Trust)** - 40%
   - T1_authoritative (1.0): 정부/공식 기관
   - T2_expert (0.95): 국제적 권위 인정 전문가
   - T3_professional (0.85): 업계 전문가
   - T4_community (0.70): 커뮤니티/UGC

2. **데이터 검증 가능성 (Verifiability)** - 25%
   - 1차 출처 (1.0): 직접 발행
   - 2차 출처 (0.7): 인용/재가공
   - 익명/불명 (0.3): 출처 불분명

3. **업데이트 일관성 (Update Consistency)** - 15%
   - RSS 자동화 (1.0): 실시간 업데이트
   - 정기 업데이트 (0.8): 주간/월간 일정
   - 비정기 (0.5): 업데이트 빈도 불규칙

4. **콘텐츠 품질 (Content Quality)** - 12%
   - 통계/데이터 기반 (1.0): 정량적
   - 전문가 분석 (0.9): 정성적 전문성
   - 뉴스/리뷰 (0.7): 시의성 중심
   - 프로모션 (0.4): 상업적 목적

5. **접근 가능성 (Accessibility)** - 8%
   - 완전 공개 (1.0): 무료, 제약 없음
   - 제한적 공개 (0.6): 일부 페이월
   - 구독 필수 (0.3): 전체 페이월

### 종합 신뢰도 점수 계산
```
Reliability Score = (Institutional × 0.4) + (Verifiability × 0.25) +
                   (Consistency × 0.15) + (Quality × 0.12) + (Accessibility × 0.08)
```

**등급 분류:**
- **A+ (0.90-1.00)**: 최고 신뢰도 - 즉시 사용 권장
- **A (0.80-0.89)**: 높은 신뢰도 - 우선 사용 권장
- **B (0.70-0.79)**: 중간 신뢰도 - 검증 후 사용
- **C (0.60-0.69)**: 낮은 신뢰도 - 보조 자료로만 사용
- **D (0.00-0.59)**: 매우 낮음 - 사용 비권장
