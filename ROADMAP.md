# WineRadar 품질 개선 로드맵

> 와인 시장 데이터 수집 품질 향상을 위한 실행 계획

## 현황 요약

| 지표 | 현재 값 | 상태 |
|------|--------|------|
| 소스 수 | 45+ | 양호 |
| 매칭률 | 43% | 개선 필요 |
| 테스트 수 | 42개 | 우수 |

---

## P0: 긴급 (없음)

현재 긴급 수정이 필요한 사항 없음. 시스템 건강 상태 양호.

---

## P1: 단기 개선 (1-2주)

### 와인 특화 키워드로 매칭률 개선

**목표**: 매칭률 43% → 70% 이상 달성

**작업 내용**:
- [ ] 와인 품종별 키워드 확장 (카베르네, 피노누아, 샤르도네 등)
- [ ] 와인 지역 키워드 추가 (보르도, 부르고뉴, 나파밸리, 토스카나 등)
- [ ] 와인 등급/분류 키워드 추가 (AOC, DOC, DOCG, Grand Cru 등)
- [ ] 양조 관련 키워드 추가 (빈티지, 테루아르, 타닌, 숙성 등)
- [ ] 한국어 와인 용어 추가 (레드와인, 화이트와인, 내추럴와인 등)

**검증 기준**:
- 매칭률 70% 이상 달성
- 기존 테스트 모두 통과
- 오탐(false positive) 비율 10% 미만 유지

---

## P2: 중기 개선 (3-4주)

### 소믈리에 리뷰 소스 추가

**목표**: 전문가 평가 데이터 확보

**작업 내용**:
- [ ] 와인21 소믈리에 칼럼 수집기 추가
- [ ] 와인폴리 한국어 리뷰 수집기 추가
- [ ] 국내 와인 블로거/인플루언서 RSS 추가
- [ ] 와인 전문 매거진 RSS 연동

### 와인 평점 통합

**목표**: 외부 평점 데이터 연동

**작업 내용**:
- [ ] Vivino API 연동 조사 및 구현
- [ ] Wine-Searcher 가격/평점 데이터 연동
- [ ] Robert Parker 점수 매핑 테이블 구축
- [ ] Wine Spectator 점수 연동 검토

**검증 기준**:
- 신규 소스 최소 10개 추가
- 소스 성공률 95% 이상 유지
- 평점 데이터 커버리지 50% 이상
